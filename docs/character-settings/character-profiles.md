## 商朝阵营

### 帝辛（纣王） ★★★★★
**基本信息**
- 本名：子受
- 在位：约公元前1075-1046年
- 谥号：纣（残义损善曰纣）
- 身份：商朝第31位君主

**性格特征**
- **复杂性**：并非单纯的暴君，早期有雄才大略
- **勇武过人**：史载"材力过人，手格猛兽"
- **刚愎自用**：拒绝纳谏，独断专行
- **骄奢淫逸**：后期沉迷享乐，荒废朝政
- **敏感多疑**：对宗室大臣充满猜忌

**关键事件**
1. 早期征伐东夷，扩展疆土
2. 修建鹿台、钜桥，耗费民力
3. 创设炮烙之刑，残害忠良
4. 囚禁西伯姬昌，后又释放
5. 杀比干、囚箕子，逼走微子
6. 牧野之战败北，鹿台自焚

**人物弧光**
- 起点：有为之君 → 转折：权力腐蚀 → 堕落：暴虐无道 → 终点：悲剧结局
- 内心矛盾：明知商朝危殆，但不愿承认错误
- 临终反思：有悔恨但保持帝王尊严

### 妲己 ★★★★☆
**基本信息**
- 出身：有苏氏部落（今河南温县）
- 身份：纣王宠妃，封后
- 传说：狐妖附身（小说可保留神秘色彩）

**历史与传说**
- **历史事实**：有苏氏献女求和，妲己入宫
- **文学形象**：祸国妖妃，助纣为虐
- **现代视角**：男权社会下的替罪羊

**小说处理**
- **双重性**：表面妖媚，内心复杂
- **动机**：部落生存、个人野心、对纣王的复杂感情
- **结局**：武王命斩首，头颅悬挂示众

### 比干 ★★★★☆
**基本信息**
- 身份：纣王叔父，商朝少师
- 称号：亘古忠臣，文财神
- 特点：七窍玲珑心

**性格特征**
- **忠贞不二**：明知必死仍强谏
- **智慧过人**：善于治国理政
- **刚直不阿**：不懂变通，宁折不弯

**关键场景**
1. 多次劝谏纣王，渐失信任
2. 鹿台夜宴当众批评纣王
3. 纣王怒曰："吾闻圣人心有七窍"
4. 剖心而死，震惊朝野

**象征意义**：忠臣的极致，道德的标杆

### 箕子 ★★★☆☆
**基本信息**
- 身份：纣王叔父，商朝太师
- 结局：佯狂为奴，后投奔周朝
- 贡献：向武王献《洪范九畴》

**性格特征**
- **大智若愚**：比干死后佯装疯癫
- **保全性命**：不同于比干的刚烈
- **文化传承**：将商朝文化带到周朝

**选择逻辑**：留得有用之身，传承文明火种

### 微子启 ★★★☆☆
**基本信息**
- 身份：纣王兄长（同母庶兄）
- 选择：出逃投周，保全宗祀
- 后世：宋国开国君主

**性格特征**
- **务实理性**：知不可为而不为
- **家族责任**：保全商朝宗庙祭祀
- **政治智慧**：选择合作而非对抗

**三仁对比**
- 比干：死谏 → 忠的极致
- 箕子：佯狂 → 智的保全  
- 微子：出逃 → 仁的延续

## 周族阵营

### 姬昌（周文王） ★★★★★
**基本信息**
- 在位：约公元前1099-1050年
- 身份：周族首领，商朝西伯
- 谥号：文（经纬天地曰文）

**仁德之君**
1. **画地为牢**：道德教化代替严刑
2. **废除人殉**：尊重生命，文明进步
3. **发展农业**：重视民生，藏富于民
4. **礼贤下士**：渭水访贤，得姜子牙

**囚禁岁月**
- 羑里七年，演算周易
- 忍辱负重，韬光养晦
- 将仇恨转化为智慧

**战略眼光**
- "三分天下有其二"仍服事商
- 表面臣服，暗中扩张
- 为武王伐纣奠定基础

### 姬发（周武王） ★★★★★
**基本信息**
- 在位：约公元前1046-1043年
- 身份：文王次子，周朝开国君主
- 谥号：武（克定祸乱曰武）

**性格成长**
- **继位初期**：年轻气盛，急于复仇
- **孟津观兵**：学会等待，成熟稳重
- **牧野之战**：英勇果决，天命所归
- **建国之后**：忧心忡忡，早逝遗憾

**关键特质**
1. **孝道**：为父报仇，完成遗愿
2. **勇气**：亲冒矢石，激励将士
3. **仁德**：不杀伯夷，安抚商民
4. **智慧**：分封诸侯，稳定天下

### 太公望（姜子牙） ★★★★★
**基本信息**
- 本名：姜尚，字子牙
- 称号：师尚父、太公望
- 年龄：出山时已七十余岁
- 身份：军师、齐國始祖

**传奇色彩**
- **渭水垂钓**：直钩钓鱼，"愿者上钩"
- **军事天才**：牧野之战战术设计
- **政治智慧**：封建制度设计者
- **文化符号**：大器晚成，智者形象

**多重身份**
1. **谋士**：提出"三分天下有其二"战略
2. **统帅**：年过七旬仍冲锋在前
3. **政治家**：协助制定周朝制度
4. **传奇人物**：封神榜主角，半人半神

### 周公旦 ★★★★☆
**基本信息**
- 身份：文王四子，武王之弟
- 贡献：制礼作乐，奠定周朝八百年基业
- 评价：儒家元圣，孔子偶像

**核心事迹**
1. **武王托孤**：受命辅佐幼主
2. **平定三监**：东征三年，稳定天下
3. **营建洛邑**：建立东都，控制东方
4. **制礼作乐**：创建周朝文化制度
5. **还政成王**：不恋权位，千古楷模

**性格特点**
- **忠诚**：鞠躬尽瘁，死而后已
- **智慧**：政治制度设计大师
- **勤勉**："一沐三捉发，一饭三吐哺"
- **谦逊**：功成身退，北面称臣

### 召公奭 ★★★☆☆
**基本信息**
- 身份：文王庶子，燕国始祖
- 贡献：与周公共理朝政
- 特点：仁德爱民，甘棠遗爱

**历史形象**
- **地方治理**：巡行乡邑，在甘棠树下办公
- **司法公正**：决狱政事，百姓无怨
- **与周公关系**：合作中有制衡，良性竞争
