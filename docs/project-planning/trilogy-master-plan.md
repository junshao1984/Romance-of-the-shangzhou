# 《商周演义》三部曲总体规划

## 作品定位
- **类型**：历史演义小说三部曲
- **总规模**：120章，约120-150万字（每部40章，每章约1-1.25万字）
- **时间跨度**：从商朝廪辛时期到周朝建立（约公元前12-11世纪）
- **地理范围**：从中原到西陲，华夏文明核心区

## 三部曲结构（每部40章，共120章）

### 第一部：《姬历传》（古公亶父）
- **主角**：姬历（古公亶父）
- **副标题**：待定（如《质子风云》《西陲崛起》）
- **篇幅**：40章，约40-50万字
- **时间**：廪辛八年 → 武乙时期
- **核心事件**：
  1. 殷都质子生涯（第1-5章）
  2. 东莘解围，收服嬴秋（第6-10章）
  3. 盘龙城支线展开（第11-15章）
  4. 归家途中经历（第16-20章）
  5. 父兄遇害，南迁岐山（第21-25章）
  6. 建立基业，周族崛起（第26-35章）
  7. 巩固势力，为子铺路（第36-40章）
- **主题**：家族生存、个人成长、地方崛起
- **历史意义**：周族从边缘部落到地方强权的转变

### 第二部：《姬昌传》（周文王）
- **主角**：姬昌（文王）
- **副标题**：待定（如《羑里之囚》《三分天下》）
- **篇幅**：40章，约40-50万字
- **时间**：武乙晚期 → 帝辛早期
- **核心事件**：
  1. 继位周伯，推行仁政（第1-8章）
  2. 被谗囚羑里，演算周易（第9-16章）
  3. 归周访贤，得姜子牙（第17-24章）
  4. 扩张势力，三分天下有其二（第25-32章）
  5. 内政外交，奠定基础（第33-36章）
  6. 去世托孤，传承大业（第37-40章）
- **主题**：仁德治国、韬光养晦、天命所归
- **历史意义**：周族从地方强权到天下共主的准备

### 第三部：《姬发传》（周武王）
- **主角**：姬发（武王）
- **副标题**：待定（如《牧野鹰扬》《天命维新》）
- **篇幅**：40章，约40-50万字
- **时间**：帝辛中后期
- **核心事件**：
  1. 继位周王，整顿内政（第1-6章）
  2. 孟津观兵，试探诸侯（第7-12章）
  3. 等待时机，积蓄力量（第13-18章）
  4. 纣王暴政，民心背离（第19-24章）
  5. 牧野之战，武王伐纣（第25-32章）
  6. 周朝建立，分封诸侯（第33-36章）
  7. 武王早逝，周公摄政（第37-40章）
- **主题**：革命正义、王朝更替、天下初定
- **历史意义**：商周革命完成，周朝八百年基业开端

## 人物传承关系

### 家族谱系
```
古公亶父（姬历，第一部主角）
└─ 季历（历史原型，可能合并或文学处理）
   └─ 姬昌（文王，第二部主角）
      └─ 姬发（武王，第三部主角）
```

### 关键人物贯穿三部
1. **胶鬲**：奴隶→巨贾→周朝功臣（贯穿三部）
2. **嬴秋后代**：秦国起源线索
3. **子鸢/有莘氏**：周族联姻关系
4. **盘龙城势力**：南方经济支持
5. **贞人集团**：商朝保守势力代表

## 历史时间线整合

### 商王序列对应
```
第一部（姬历）：
廪辛（8年）→ 康丁 → 武乙 → 文丁早期

第二部（姬昌）：
文丁中后期 → 帝乙 → 帝辛早期

第三部（姬发）：
帝辛中后期 → 周朝建立
```

### 关键年代节点
- **廪辛8年（约前12世纪）**：姬历质子故事开始
- **武乙时期**：姬历南迁岐山，周族崛起
- **文丁时期**：季历（？）扩张被杀，姬昌继位
- **帝乙时期**：姬昌被囚羑里
- **帝辛早期**：姬昌归周，扩张势力
- **帝辛33年（前1046）**：牧野之战，商朝灭亡

## 主题演进

### 第一部主题
- **生存与成长**：个人和家族在乱世中的生存智慧
- **地方崛起**：边缘部落如何成长为地方强权
- **早期商周关系**：质子制度下的控制与反控制

### 第二部主题
- **仁德治国**：文王仁政与商朝暴政的对比
- **韬光养晦**：在强大中央政权下的生存发展策略
- **天命观念**：从"天命玄鸟"到"周虽旧邦"的思想转变

### 第三部主题
- **革命正义**：武王伐纣的合法性与正义性
- **王朝更替**：历史循环与文明进步
- **天下秩序**：周朝封建制度与新天下秩序建立

## 叙事风格演进

### 第一部风格
- **个人史诗**：聚焦姬历个人成长
- **多线叙事**：殷都、东莘、盘龙城等多地故事
- **生存挣扎**：强调在逆境中的奋斗

### 第二部风格
- **政治智慧**：展现文王的政治谋略
- **文化深度**：融入周易哲学、礼乐文化
- **战略博弈**：商周之间的政治军事博弈

### 第三部风格
- **历史高潮**：牧野之战的宏大叙事
- **天命论证**：探讨王朝更替的历史合法性
- **制度创建**：周朝封建礼乐制度的建立

## 创作衔接点

### 第一部到第二部过渡
